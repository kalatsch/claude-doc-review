claude-doc-review

Turn any markdown document Claude writes into a human-friendly review page — clear prose, glossary tooltips, collapsible technical detail — with an inline text-selection commenting layer that syncs live to Claude. No cloud, no accounts, no telemetry.

/doc-review inline comments & replies glossary tooltips collapsible detail light / dark 100% local
Install

Install from this repo

Inside Claude Code — no central publishing involved. These are two separate slash commands: run them one at a time, in order.

1Add this GitHub repo as a plugin source
/plugin marketplace add kalatsch/claude-doc-review
2Install the plugin from it
/plugin install claude-doc-review@claude-doc-review

↻Updating later? No commands needed — open the built-in /plugin menu, go to Manage plugins → claude-doc-review → Update. (Restart the window, or run /reload-plugins, to load the new version into the current session.)

After install

Usage

  • /doc-review path/to/doc.md — humanize, serve & open
  • Select text on the page → leave a comment
  • Claude answers right in the thread
  • Resolve threads when settled

What it does

  • Rewrites the doc into plain prose + a Словарь glossary
  • Deep detail folds into collapsible blocks
  • Output lands in .claude/doc-review/<slug>/
  • Auto-ignored from your project's git
